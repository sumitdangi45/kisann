# 📅 Smart Reminders - Complete Guide

## Overview
Smart Reminders is the **MOST IMPORTANT** feature that helps farmers track their entire crop lifecycle with automated reminders for every task. Farmers can upload photos to monitor crop health and get disease alerts.

## 🎯 Key Features

### 1. **Crop Registration**
- Farmer enters: Crop name, planting date, field name, area
- System automatically creates a complete task schedule for the entire crop lifecycle
- Example: "Maine moong lagayi hai" → System creates 12 reminders for 90 days

### 2. **Automated Task Schedule**
Each crop has a predefined lifecycle with tasks at specific days:
- **Day 0**: Prepare field and sow seeds
- **Day 3**: Water the field
- **Day 7**: First weeding
- **Day 14**: Apply nitrogen fertilizer
- **Day 21**: Second weeding
- **Day 30**: Check for pests and diseases
- ... and so on until harvest

### 3. **Smart Reminders**
- Reminders are scheduled based on planting date
- Each reminder shows:
  - Task description
  - Scheduled date
  - Priority level (High/Medium/Low)
  - Task type (Irrigation, Fertilizer, Pest Control, etc.)

### 4. **Task Completion Tracking**
- Farmer marks tasks as complete
- Can add notes about what was done
- Progress bar shows completion percentage
- System tracks which tasks are done and pending

### 5. **Crop Photo Monitoring**
- Upload photos at any time
- System automatically analyzes photos for diseases
- Shows health status (Healthy/Disease detected)
- Creates a photo timeline to track crop growth

### 6. **Multi-Crop Management**
- Manage multiple crops simultaneously
- Each crop has its own schedule and photos
- Track progress for each field separately

## 📱 How to Use

### Step 1: Add a Crop
1. Go to **Smart Reminders** page (`/reminders`)
2. Click **"+ Add Crop"** button
3. Fill in:
   - **Crop Name**: Select from dropdown (rice, wheat, moong, maize, cotton, potato, tomato)
   - **Planting Date**: When you planted the crop
   - **Field Name**: Name of your field (e.g., "Field 1", "North Field")
   - **Area**: Size in acres
4. Click **"Add Crop & Create Schedule"**
5. System creates all reminders automatically!

### Step 2: View Your Crops
- All your crops appear as cards
- Each card shows:
  - Crop name
  - Field location
  - Area in acres
  - Days elapsed since planting
  - Task completion progress (%)
  - Number of photos uploaded

### Step 3: Check Reminders
1. Click on a crop card to select it
2. Go to **"📋 Reminders"** tab
3. See all tasks with:
   - Task description
   - Scheduled date
   - Priority color (Red=High, Yellow=Medium, Blue=Low)
   - Task type icon (💧=Water, 🧪=Fertilizer, 🐛=Pest Control, etc.)

### Step 4: Complete Tasks
1. When you complete a task, click **"✓ Done"** button
2. Optionally add notes about what you did
3. Task is marked as complete
4. Progress bar updates automatically

### Step 5: Upload Crop Photos
1. Go to **"📸 Photos"** tab
2. Click **"Upload Photo"** button
3. Select a photo from your phone/camera
4. Add optional notes (e.g., "Leaves turning yellow")
5. System analyzes the photo for:
   - Disease detection
   - Health status
   - Shows results immediately

### Step 6: Monitor Progress
- View progress bar for each crop
- See completion percentage
- Track photos uploaded
- Monitor crop health over time

## 🌾 Supported Crops & Lifecycles

### 1. **Rice** (120 days)
- Season: Monsoon/Winter
- Key tasks: Weeding, fertilizer, pest control, monitoring
- Critical stages: Panicle initiation, flowering, grain filling

### 2. **Wheat** (150 days)
- Season: Winter
- Key tasks: Weeding, fertilizer, disease monitoring
- Critical stages: Tillering, stem elongation, heading, flowering

### 3. **Moong** (90 days) ⭐ Most Popular
- Season: Summer/Monsoon
- Key tasks: Weeding, fertilizer, pest control
- Critical stages: Flowering, pod formation, pod maturation

### 4. **Maize** (120 days)
- Season: Summer/Monsoon
- Key tasks: Weeding, fertilizer, pest control
- Critical stages: Tasseling, silking, grain filling

### 5. **Cotton** (180 days)
- Season: Summer
- Key tasks: Weeding, fertilizer, pest control
- Critical stages: Flowering, boll formation, boll maturation

### 6. **Potato** (90 days)
- Season: Winter
- Key tasks: Earthing up, fertilizer, pest control
- Critical stages: Tuber formation, tuber bulking

### 7. **Tomato** (120 days)
- Season: Summer/Winter
- Key tasks: Pruning, staking, pest control
- Critical stages: Flowering, fruit setting, fruit development

## 🔔 Priority Levels

- **🔴 HIGH (Red)**: Critical tasks - must be done on time
  - Examples: Watering during critical stages, disease spraying
  
- **🟡 MEDIUM (Yellow)**: Important tasks - should be done on time
  - Examples: Regular weeding, fertilizer application
  
- **🔵 LOW (Blue)**: Routine tasks - can be done with some flexibility
  - Examples: General monitoring, notes

## 📸 Photo Analysis

When you upload a photo, the system:
1. **Analyzes the image** for diseases
2. **Detects crop health status**:
   - ✅ **Healthy**: No diseases detected
   - ⚠️ **Disease**: Specific disease identified
3. **Shows results immediately**
4. **Stores photo timeline** for tracking growth

## 💡 Tips & Best Practices

1. **Add crops immediately after planting** - Don't wait, add crop on day 0
2. **Check reminders daily** - Set a habit to check app every morning
3. **Upload photos regularly** - At least once a week for disease monitoring
4. **Add notes** - Help yourself remember what you did
5. **Mark tasks complete** - Keep progress bar accurate
6. **Use voice reminders** - Click 🔊 to hear task description
7. **Monitor photos** - Watch for disease patterns over time
8. **Plan ahead** - Check upcoming reminders for next week

## 🔌 API Endpoints

### Add Crop
```
POST /api/reminders/add-crop
Body: {
  "farmer_id": "default_farmer",
  "crop_name": "moong",
  "planting_date": "2026-04-18",
  "field_name": "Field 1",
  "area_acres": 1
}
```

### Get Farmer's Crops
```
GET /api/reminders/crops/{farmer_id}
```

### Get Today's Reminders
```
GET /api/reminders/today/{crop_id}
```

### Get Upcoming Reminders
```
GET /api/reminders/upcoming/{crop_id}?days=7
```

### Mark Reminder Complete
```
POST /api/reminders/complete
Body: {
  "reminder_id": "...",
  "notes": "Task completed successfully"
}
```

### Upload Photo
```
POST /api/reminders/upload-photo/{crop_id}
Form Data: {
  "file": <image_file>,
  "notes": "Crop looking healthy"
}
```

### Get Crop Photos
```
GET /api/reminders/photos/{crop_id}
```

### Get Crop Statistics
```
GET /api/reminders/stats/{crop_id}
```

## ❓ FAQ

**Q: Can I add a crop I planted weeks ago?**
A: Yes! Enter the actual planting date, and reminders will be calculated from that date.

**Q: What if I miss a reminder?**
A: No problem! You can still mark it as complete later. The system tracks all tasks.

**Q: Can I upload multiple photos?**
A: Yes! Upload as many photos as you want. Each photo is analyzed for diseases.

**Q: What if my crop is not in the list?**
A: Currently supported: rice, wheat, moong, maize, cotton, potato, tomato. More crops can be added.

**Q: How accurate is disease detection?**
A: Uses AI model trained on plant disease images. Accuracy depends on photo quality.

**Q: Can I manage multiple fields?**
A: Yes! Add each field as a separate crop with different field names.

**Q: What if I want to stop tracking a crop?**
A: You can mark crop status as "completed" or "abandoned" to hide it.
