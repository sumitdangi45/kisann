SCENARIO: Farmer wants to know which crop to grow

INPUT DATA:
  ├─ Nitrogen (N): 40 kg/ha
  ├─ Phosphorous (P): 40 kg/ha

  ├─ pH: 6.5
  ├─ Rainfall: 200 mm
  └─ City: Delhi

EXPECTED OUTPUT: Rice

  3. Click "Predict"

🧪 TEST 2: FERTILIZER RECOMMENDATION
────────────────────────────────────────────────────────────────────────────────

SCENARIO: Farmer has rice crop with low nitrogen

INPUT DATA:
  ├─ Crop Name: Rice
  ├─ Nitrogen (N): 20 kg/ha (LOW - should be 40)
  ├─ Phosphorous (P): 40 kg/ha
  └─ Potassium (K): 40 kg/ha

EXPECTED OUTPUT: Recommendation to add nitrogen fertilizer

STEPS:
  1. Click "Fertilizer Suggestion" on home page
  2. Select "Rice" as crop
  3. Enter N=20, P=40, K=40
  4. Click "Predict"
  5. Should see nitrogen deficiency recommendation
